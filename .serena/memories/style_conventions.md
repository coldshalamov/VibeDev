# Code Style and Conventions

## Python
- Use Type Hints where possible.
- Prefer `FastAPI` for HTTP endpoints.
- Async code is preferred (using `aiosqlite`, `asyncio`).
- MCP protocols should be strictly followed.

## TypeScript/React
- Use Functional Components with Hooks.
- **Zustand** for state management.
- **XYFlow (@xyflow/react)** for workflow graph rendering.
- Tailwind CSS for styling.
- Follow ESLint rules.
